What is DOM and What is the purpose of DOM?

DOM means Document Object Model. It is one kind of programming interface that allows create,change,remove or styles element from the document. It helps us add events to these elements to make our web page more dynamic.
We are using Dom because the developing of the DOM is to maintain a programming interface for HTML documents. JavaScript language is probably of the most popular ways of working with HTML Documents using the DOM.
